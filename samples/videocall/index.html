In-app data calling demo
Tutorial
  • Open the url in two browsers/tabs and enter two different user ids (see the documentation for more details) and press start in both browsers/tabs.
  • Enter a callee in "Make a call" on one of the browsers/tabs, should be the user id in the other browser/tab.
  • Open the console in your browser to get more insight about what is happening.
  • Note: Chrome currently supports two tab calling. If you want to call in Firefox, you will have to create two separate profiles and call between them
